在windows和linux之间传送文件


方法一：使用SCP

  1. 输入目标地址的ip和端口号即可实现连接。

  2. 直接可以在windows和linux之间互传文件。


方法二：使用SSH

1. 在使用Putty前要确保Linux的ssh service是出于启动状态，可使用service ssh status去查看是否有启动，如果ssh并未安装，可以用如下命令进行安装(示例为Ubuntu)

  启动ssh服务：sudo /etc/init.d/ssh start

  可以用 ps -e | grep ssh来查看ssh服务是否打开

  stack@TEST:/etc/init.d$ ps -e | grep ssh

  1378 ? 00:00:00 sshd

  31015 ? 00:00:00 sshd

  31066 ? 00:00:00 sshd

2. 使用Putty 登陆到Linux系统中：输入IP, 端口号为22， 使用SSH, 在弹出的cmd中输入用户名和密码登陆.

3. 下载pscp.exe(点我下载)放到c:\windows\system32目录下;

4. 打开CMD，运行命令：pscp source_path username@hostIP:target_path，如pscp C:\Users\name\Desktop\gengui.png pp@192.168.191.25:/home/pi/Desktop 即可从windows向linux发送文件。
