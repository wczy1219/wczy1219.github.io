如何设置vim配色方案


1. 输入sudo su，以root权限登录


2. 备份vimrc: cp /etc/vim/vimrc /etc/vim/vimrc.bak


3. 编辑vimrc文件：vim /etc/vim/vimrc


4. 在最后加入下列代码：

set fenc=utf-8 "设定默认解码为UTF8
set nocompatible "不要使用vi的键盘模式,而是vim自己的
set t_Co=256 "设置vim显示颜色为256色
set smarttab "在行和段开始处使用制表符
set tabstop=4 "制表符统一为4
set softtabstop=4
set shiftwidth=4
set autoindent "自动缩进
set number "显示行号
set so=10 "当光标离窗口上下边界的最小行距为10时自动滚动
set nowrap "不换行
set history=100 "保存100行历史命令行
set viminfo+=! "保存全局变量
set nobackup "不备份文件
set linespace=0 "字符间插入的像素行数为0
set ruler "在状态行上显示光标所在位置的行号和列号
set cmdheight=2 "命令行的高度为2
set showmatch "高亮显示匹配的括号
set ignorecase "搜索时忽略大小写
syntax on "语法高亮
filetype on "侦测文件类型
filetype plugin on"载入文件类型插件
