1)通过下方命令可安装vim

  • sudo apt-get install vim

2)通过下方命令先拷贝一份配置文件到用户目录下:

  • cp /usr/share/vim/vimrc ~/.vimrc

3)编辑配置文件

  • vi ~/.vimrc

4)下方是可选的配置以及所代表的含义：

  • syntax on               '' 语法高亮
  • set nu                  '' 设置行号
  • filetype on             '' 检测文件的类型
  • set history=1000        '' 记录历史记录的行数
  • set background=dark     '' 背景使用黑色
  • set autoindent          '' 自动缩进
  • set cindent             '' 针对C语言语法的自动缩进
  • set smartindent         '' 依据上面的对齐格式，智能地选择对齐方式，对于类似C语言编写上有用
  • set tabstop=4           '' 设置tab键为四个空格
  • set shiftwidth=4        '' 设置当航之间交错时使用四个空格
  • set ai                  '' 设置自动缩进
  • set showmatch           '' 设置匹配模式，类似当输入一个左括号时会匹配相应的右括号
  • set ruler               '' 编辑模式下，右下角显示光标位置的状态行
