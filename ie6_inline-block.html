IE6中实现inline-block 1)inline元素直接使用display:inline-block即可。 2)block元素可采用以下两种方法进行实现。 方法一：先用inline-block触发块元素，然后再定义inline让块元素呈递为内联对象。 
div{
    display:inline-block;
}
div{
    display:inline;
}
方法二：直接让块元素呈递为内联对象，然后触发元素的layout： 
div{
    display:inlune-block;
    _zoom:1;
    _display:inline;
}
