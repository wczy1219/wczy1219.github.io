解决Chrome浏览器提示“Adobe flash player不是最新版本”

1）访问以下页面：http://www.adobe.com/support/flashplayer/debug_downloads.html

  • sudo apt-get install nautilus-open-terminal

2）找到下方链接并点击下载文件：

  • Download the Flash Player content debugger for Opera and Chromium based applications – PPAPI

3）下载完成后安装并重启浏览器即可。
